# PetCare web version

## How to run
<ul>
    <li>Create you virtual environment</li>
    <li>Start your virtual environment</li>
    <li>
        Install all dependencies
        <code>pip install -r requirements.txt</code>
    </li>
    <li>
        Run the PetCare
        <code>python3 manage.py runserver</code>
    </li>
</ul>
